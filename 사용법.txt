========================================
게이트맨 웹사이트 - 로컬 서버 사용법
========================================

1. 서버 시작
   - START_SERVER.bat 더블클릭
   - 브라우저에서 http://localhost:3000/index.html 접속

2. 서버 종료
   - STOP_SERVER.bat 더블클릭
   - 또는 서버 창에서 Ctrl+C

3. 오류 발생 시 (EADDRINUSE)
   - STOP_SERVER.bat 실행
   - 다시 START_SERVER.bat 실행

========================================
헤더/푸터 수정 방법
========================================

헤더 수정:
  includes/header.html 파일 수정
  → 모든 25개 페이지에 자동 반영!

푸터 수정:
  includes/footer.html 파일 수정
  → 모든 25개 페이지에 자동 반영!

========================================
실제 웹 호스팅 배포
========================================

GATEMAN 폴더 전체를 서버에 업로드하면
자동으로 작동합니다!

- 구글 Firebase Hosting
- GitHub Pages
- Netlify
- 일반 웹 호스팅 (FTP)

서버에 올리면 localhost:3000 없이
실제 도메인으로 바로 접속 가능!
